### Usage Example | مثال الاستخدام

*English:*
1. Navigate to your desired category
2. Choose language (English/Arabic)
3. Copy the prompt
4. Customize placeholders like [TOPIC], [AUDIENCE]
5. Use with your preferred AI tool

*العربية:*
1. انتقل إلى الفئة المطلوبة
2. اختر اللغة (إنجليزية/عربية)
3. انسخ البرومبت
4. خصص العناصر النائبة مثل [الموضوع]، [الجمهور]
5. استخدم مع أداة الذكاء الاصطناعي المفضلة لديك

### Sample Prompt | برومبت عينة

markdown
English:
"Act as a professional content writer. Create a comprehensive blog post about [TOPIC] that includes engaging headlines, SEO optimization, and actionable takeaways for [TARGET AUDIENCE]."

Arabic:
"تصرف ككاتب محتوى محترف. اكتب مقالاً شاملاً حول [الموضوع] يتضمن عناوين جذابة وتحسين لمحركات البحث ونصائح قابلة للتطبيق لـ [الجمهور المستهدف]."


## 🛠 Best Practices | أفضل الممارسات

<details>
<summary><strong>🇺🇸 English Best Practices</strong></summary>

### Prompt Engineering Tips:
- *Be Specific*: Use clear, detailed instructions
- *Provide Context*: Give relevant background information
- *Set Constraints*: Define length, format, and tone requirements
- *Use Examples*: Include sample inputs and desired outputs
- *Iterate*: Refine prompts based on AI responses
- *Test Thoroughly*: Validate prompts across different scenarios

### Structure Template:

Role: [WHO should the AI act as]
Task: [WHAT should be accomplished]
Context: [BACKGROUND information]
Format: [HOW should output be structured]
Constraints: [LIMITATIONS and requirements]
Examples: [SAMPLE inputs/outputs]


</details>

<details>
<summary><strong>🇸🇦 أفضل الممارسات بالعربية</strong></summary>

### نصائح هندسة البرومبتات:
- *كن محدداً*: استخدم تعليمات واضحة ومفصلة
- *قدم السياق*: أعط معلومات خلفية ذات صلة
- *ضع قيود*: حدد متطلبات الطول والتنسيق والنبرة
- *استخدم أمثلة*: أدرج عينات من المدخلات والمخرجات المرغوبة
- *كرر التحسين*: حسن البرومبتات بناءً على استجابات الذكاء الاصطناعي
- *اختبر جيداً*: تحقق من البرومبتات عبر سيناريوهات مختلفة

التنسيق: [كيف يجب هيكلة الناتج]
القيود: [القيود والمتطلبات]
أمثلة: [عينات مدخلات/مخرجات]
